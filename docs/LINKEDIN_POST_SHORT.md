## Lessons Learned

1. Self-improvement is hard — Requires safeguards (debate, monitoring, verification, rollback)

2. Memory is everything — Stateless agents are like goldfish. Persistent memory transforms the experience.

3. Documentation writes itself — With the right system (LLM wiki), it becomes a byproduct of operation.

4. Multi-agent > Single agent — Specialized agents coordinating beat one generalist.

5. Local-first wins — Privacy, speed, cost savings. Cloud APIs are overrated for most tasks.

6. UX matters — Even the most advanced AI is useless if people can't use it.
